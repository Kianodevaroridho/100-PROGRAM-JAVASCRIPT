PROGRAM GABUNGKAN DUA ARRAY

Array 1: [1, 2, 3]

Array 2: [4, 5, 6]

Hasil :

const BtnGabung = document.getElementById("btn-gabung") BtnGabung.addEventListener("click", function(){ const arr1 = [1, 2, 3]; const arr2 = [4, 5, 6]; const gabungan = arr1.concat(arr2); document.getElementById('hasil').textContent = gabungan.join(', '); })
